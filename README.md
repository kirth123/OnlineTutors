# Online Tutors

![image](https://github.com/user-attachments/assets/3cfbe558-b8bb-4550-903a-ca5855d2630c)


This website allows tutors and students to talk to each other over video chat and draw on a collaborative whiteboard. It uses React, Node.js, MongoDB, Zoho Mail, and an AWS EC2 instance. You can visit the site at https://onlinetutorstoday.net.
The site uess the WebRTC protocol for video connections and WebSockets for the whiteboard feature.

## Testing
You can run the unit tests for the React components by running these commands (after cloning my repo).

```
cd frontend
npm install
npm test 
```
